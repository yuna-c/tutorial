Git

  • Git 설치
  • Git 설치(Mac)
  • cd desktop
  • mkdir 생성폴더명
  • code . vscode로 폴더 열기
        
$ git init : 빈 레퍼지토리 생성
$ git add . : 변경 사항 기록
$ git status : 프로젝트 어떻게 관리 되고 있는지 확인
$ git config --global user.name '사용자명'
$ git config --global user.email '이메일@주소'
$ git config user.name 확인
$ git config user.email 확인
$ git remote add origin https://github.com/yuna-c/Git-tutorial.git 저장소 등록
$ git commit -m "분기점" 등록
$ git log 아이디값 확인
clear 터미널 초기화
$git checkout 원하는 커밋 시점 돌아가기
$git checkout master 최신커밋 돌아가기
$git branch 브랜치명 가지뻗기
$git branch -v 브랜치 버전
$git push origin master 마스터브랜치 올리기
$git push origin --all 모든 브랜치 다 올리기
$git clone https://github.com/yuna-c/Git-tutorial.git
$git branch -a 클론 후 모든 브랜치 보기

1. 원격 저장소에 복수개의 브랜치가 있는 것 확인
2. 바탕화면에 home 폴더를 새로 생성하고 해당 폴더로 이동
3. 해당 폴더 안쪽에 원격 저장소 클론
4. 클론한 폴더로 이동해서 vscode로 열기 (cmd)
5. 클론한 폴더의 숨긴 브랜치까지 확인하고 서브 브렌치로 이동 

$git checkout origin/rosebox 하위 브랜치 가지고오기
$git checkout rosebox 하위 브랜치로 들어가기
$git branch -v 브랜치 확인하기

$git pull origin master 최신버전 동기화

$git remote remove origin 원격저장소에서 다운받고나서 주소 초기화
$git push --mirror https://github.com/yuna-c/home.git
$git remote add origin https://github.com/yuna-c/home.git 다시 리모트
